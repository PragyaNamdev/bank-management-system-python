# 🏦 Bank Management System (Python)

This is a simple command-line based Bank Management System built using Python.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- View Details
- Update Details
- Delete Account

## 🛠 Tech Used
- Python
- JSON (for data storage)

## 📌 How to Run
1. Clone the repository
2. Run:
   python main.py
